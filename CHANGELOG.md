# Changelog

All notable changes to this project will be documented in this file.

## [1.0.5] - 2026-05-06

### Fixed
- Forced version bump to resolve pub.dev publishing conflicts.
- Final verification of the "Headless" plugin architecture.

## [1.0.4] - 2026-05-06

### Changed
- Total cleanup of native code. Deleted all unused Activities, Layouts, and Resources.
- Finalized "Headless" architecture: Native code is now exclusively for NFC bridge logic.
- Production-ready stable release.

## [1.0.3] - 2026-05-06

### Removed
- Removed all native Android Activities to prevent UI/Manifest conflicts with Flutter host apps.

## [1.0.2] - 2026-05-06

### Improved
- Synchronized Android and iOS project configurations.
- Improved build compatibility with modern Flutter environments.

## [1.0.1] - 2026-05-06

### Fixed
- Fixed Android "Main Class Not Found" error.
- Fixed iOS directory naming and structure.

## [1.0.0] - 2026-05-05

### Added
- Initial release of NFC Pro Manager.
